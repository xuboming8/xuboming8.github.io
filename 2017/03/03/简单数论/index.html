UVA 12716

题目要求

输入整数n（1≤n≤30000000）有多少对整数（a，b）满足：1≤b≤a≤n 且gcd(a，b)=a XOR b
例如n=7时 有4对：（3，2）（5，4）（6，4）（7，6）

参考AC代码

思路

我们在打印出一些三元组（a，b，c）后可以发现c=a-b
有了这个结论 我们枚举c和a 计算出b 验证是否满足题意
枚举c后 由于c是a的约数 所以a只需依次加上c即可

UVA 1635

题目要求

给定n个数a1,a2····an,依次求出相邻两个数值和，将得到一个新数列，重复上述操作，最后结果将变为一个数，问这个数除以m的余数与那些数无关？例如n=3,m=2时
第一次得到a1+a2,a2+a3，在求和得到a1+2*a2+a3，它除以2的余数和a2无关。1=<n<=10^5, 2=<m<=10^9

参考AC代码

思路

1、首先我们可以发现对于给定的n其实每项的系数就是C(n-1,i-1)，所以我们只需要找到每项的系数对m取余是否为0即可
2、由于m的取值范围为10^9,所以我们只需要筛选 √(10^9)的素数，然后对m进行分解；如果分解后m>1，说明当前m的是原m的一个素数，而且m> √(10^9)，因此我们只
需记录它即可
3、根据C(n,k)=C(n,k)(n-i+1)/i;(根据这个公式我们可以发现C(n,k)(n-i+1)一定能整除i)，对m分解式中的素数进行操作即可，具体见代码
getprime函数打印出素数表
init函数求出m的唯一分解定理中的所有质数和指数
get_factor函数对每个组合数进行如下操作：是否可以被每个质因数整除 若指数除为0表示整除完
因为存在分子分母的约分关系 所以x应减 y应加
另外也需要注意输出格式

UVA 10820

题目要求

求1-n之间有多少对互质的数

参考AC代码

思路

裸欧拉函数变形求解
使用O（nloglogn）算法求解
除了（1，1） 其余求出的均为x＜y的情况 最后应×2 所以结果为2×ans+1
ans=phi(2)+phi(3)+···+phi(n)
注：欧拉函数的公式比变形复杂 但是求解要快 要注意根据时间选择合适的算法

UVA 1262

题目要求

给出两个6行5列的字母矩阵，一个密码满足：密码的第i个字母在两个字母矩阵的第i列均出现。
然后找出字典序为k的密码，如果不存在输出NO

参考AC代码

思路

回溯法更容易写出
map三维数组可以存入两个矩阵
cnt记录是不是字典序第k个
对每一列 对出现的字母置为1 对两个矩阵中该列均为1的元素进行记录并回溯

hackerrank

题目要求

给出数组A和数组B 长度均为n
要从从数组A中选出x 从数组B中选出y 满足gcd(x,y)最大 输出x+y

参考AC代码

思路

对2个数组预处理出某个因数倍数的最大值
再从最大值开始逆推 第一个满足条件即为所求
